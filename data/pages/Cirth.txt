
== Usage ==
This template returns a Wikipedia article name for an alpha-4 script code:

{{ISO 15924/wp-name|Cirt}} → Cirth so the link is: Cirth
Note: when resolving red links or wrong links, please edit this {{ISO 15924/wp-name}} or {{ISO 15924/wp-category}}. This is where the connection between ISO code and a Wikipedia article is made. You might also consider using a redirect. (We cannot change ISO 15924 itself.)
